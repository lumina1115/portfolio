## 一、设计调研

> 痉挛性脑瘫发病率 2‰–3.4‰，痉挛型占 75%。患儿肌肉张力紊乱 → 足内翻 → 步态不稳 → 跌倒风险高。但现有产品无一触及"病耻感"这个真正的隐形伤口。

![传统医疗支具与本设计的竞品象限对比](https://portfolio-image-cdn.zongxz66.workers.dev/project-1/competitor-comparison.jpg)

**三层用户痛点**

| 痛点 | 真实声音 |
|---|---|
| 足内翻 | "走路看起来总是走不稳，很怕他摔倒" |
| 沟通障碍 | 患儿无法清晰表达不适，哭闹被误读为"闹情绪" |
| 病耻感 | "不想穿着它去上学，老师同学都会特别注意我" |

## 二、设计定位

> 让康复，始于日常的每一步。

**产品名 Gradient**（渐变）

```
足内翻   ──→  AFO 三点力学矫形
沟通障碍  ──→  智能压力传感监测模块
病耻感   ──→  模块化 × 时尚运动外观
```

- **情绪板关键词**：科技感 · 轻量康复 · 温和治愈
- **核心功能三角**：矫正 × 监测 × 温度（情感化）
- **动线重组**：打破繁琐穿戴，重构 **"穿-套-贴-联-走" 5 步极简产品流程**——脚后跟贴合最深处 → 绑带紧固 → 卡扣锁定 → LED 灯光闪烁确认启动 → 开始训练

## 三、设计展开 —— 怎么做出来的？

> 将微型智能硬核压进极薄的鞋底夹层，通过代码与传感器捕捉冰冷数据背后的行走温度。

![Arduino 与 FSR 传感器电路原理](https://portfolio-image-cdn.zongxz66.workers.dev/project-1/circuit-schematic.jpg)
![电子元器件鞋底堆叠爆炸图](https://portfolio-image-cdn.zongxz66.workers.dev/project-1/exploded-view.jpg)

**技术选型**
- AFO 三点力系统原理（F₁踝底 / F₂跟骨后 / F₃小腿前）—— 引用学术文献（朱莹，东华大学，2025）
- 柔性传感面料 **ThermoFit**：超材料热成型 + 传感嵌入，实现人体贴合与步态监测（参考 ACM IMWUT，2023）
- 磁吸机械锁卡扣：借鉴 MagSafe + 摄影器材快装板逻辑，连接清晰、拆装便捷

**Arduino 压力传感技术原型自主搭建**

以压电传感器 + Arduino 开发板 + 面包板搭建传感回路，分两阶段验证：
1. **平面手压测试**：验证传感器响应曲线的稳定性与灵敏度
2. **粘土脚型模拟测试**：将传感器嵌入手塑粘土足底模型，模拟真实行走时的足底压力分布，实时采集步态波形并输出可视化曲线

从传感布局 → 信号采集 → 波形可视化，完整跑通智能监测的技术链路。

```cpp
// 核心逻辑片段：FSR 压力差值异常与步态判定
const int FSR_INNER = A0; // 内侧传感器
const int FSR_OUTER = A1; // 外侧传感器

void setup() {
  Serial.begin(9600);
}

void loop() {
  int innerVal = analogRead(FSR_INNER);
  int outerVal = analogRead(FSR_OUTER);
  if ((innerVal - outerVal) > 350) { // 触发足内翻压力阈值
    triggerAlert(); // 隐性振动反馈与数据同步
  }
  delay(50);
}
```

**实体草模制作**

以 KT 板为骨架，手工涂抹白色腻子塑形足底轮廓，反复打磨推敲——验证形态比例与穿戴包裹感，将草图中的弧面关系落地为可触摸的三维体量。草模制作与技术原型并行推进，形态与功能双线同步迭代。

**AIGC 辅助形态发散**：在草图雏形基础上，引入 AIGC 进行高频视觉迭代；探索多组不同的流线型运动线条与网面拼接方案，快速逼近最佳形态解。

## 四、设计呈现 —— 最终结果如何？

> 外刚内柔的双层异质拼接，用蓝粉渐变的视觉张力消解医疗设备的冰冷符号。

![蓝粉渐变卡扣细节工艺](https://portfolio-image-cdn.zongxz66.workers.dev/project-1/buckle-detail.jpg)

**整体形态**：高帮运动鞋廓形，消解"医疗器械感"。主色浅蓝灰 + 蓝粉渐变装饰件（呼应 Gradient）。厚底流线型大底，兼顾缓震功能与运动视觉感。

**CMF 分层拆解**

| 区域 | 材质 | 功能 |
|---|---|---|
| 硬质外壳 | 医用硬质硅胶 | AFO 三点力矫正支撑 |
| 内衬面料 | ThermoFit 柔性传感织物 | 实时步态 + 足底压力监测 |
| 鞋舌/内里 | 拼接透气网面 | 长时穿戴舒适性 |
| 绑带 | 3 条可调节绑带（浅灰色） | 围度自适应固定 |
| 卡扣 | 蓝粉渐变磁吸机械锁扣 ×2 | 快拆快装 + 视觉亮点 |
| 大底 | 厚底缓震结构 | 步态稳定 + 流线造型 |

**自选印花**：鞋身表面设计可选卡通浮雕印花，儿童主动参与外观个性化——将医疗产品转化为"我的东西"，病耻感的解决方案落地于产品细节。

**五步使用流程**：穿（脚跟贴合最深处）→ 套（磁吸连接稳定）→ 贴（绑带 + 卡扣紧固）→ 联（LED 灯光闪烁，确认启动）→ 走（开启矫正训练）。Step 4 的 LED 灯光反馈，让患儿感知"设备在帮助我"——将"沟通障碍"痛点转化为可见的情感交互节点。
