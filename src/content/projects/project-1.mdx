## 耻感解构：从"冰冷医疗"到"潮流机能"的心理跨越

> 针对 75% 痉挛型脑瘫患儿的隐形痛点，用去医疗化的机能美学击碎传统矫形器的"耻感焦虑"。

![传统医疗支具与本设计的竞品象限对比](https://portfolio-image-cdn.zongxz66.workers.dev/project-1/competitor-comparison.jpg)

- **痛点靶向捕捉**：调研证实痉挛性脑瘫发病率达 **2‰ - 3.4‰**，患儿因肌肉张力无常导致**足内翻**与步态不稳；8 岁典型用户常刻意拉长裤脚遮挡支具，因**病耻感**隐瞒疼痛。

- **市场空白切入**：传统 **AFO 踝足矫形器**与外骨骼设备普遍呈现强医疗、低智能、低颜值特征；设计精准锁定**高智能、强机能风**的蓝海市场，让医疗器械化身为孩子眼中"很酷的运动鞋"。

- **动线无缝重组**：打破繁琐穿戴，重构 **"穿-套-贴-联-走" 5 步极简产品流程**；确保脚后跟完全贴合最深处，绑带紧固后自动触发灯光闪烁激活监测。

## 软硬交织：微型电子堆叠与 FSR 编程闭环

> 将微型智能硬核压进极薄的鞋底夹层，通过代码与传感器捕捉冰冷数据背后的行走温度。

![Arduino 与 FSR 传感器电路原理](https://portfolio-image-cdn.zongxz66.workers.dev/project-1/circuit-schematic.jpg)

![电子元器件鞋底堆叠爆炸图](https://portfolio-image-cdn.zongxz66.workers.dev/project-1/exploded-view.jpg)

- **微型架构堆叠**：选用 **Arduino Nano/Micro** 核心微控制器，配合薄膜 **FSR 压力传感器**阵列、**温度传感器**与蓝牙微型模块，在不破坏矫形鞋底厚底缓震结构的前提下，实现电子元器件的**超轻薄、高集成度埋设**。

- **力学算法映射**：依托 **AFO 三点力学矫治原理**，通过核心算法实时读取传感器数据；当检测到足底内侧压力突变或温度异常时，系统判定发生**足内翻倾向或压疮风险**，自动建立"患儿-家长-康复师"的隐性沟通桥梁。

- **硬核编程验证**：

```cpp
// 核心逻辑片段：FSR 压力差值异常与步态判定
const int FSR_INNER = A0; // 内侧传感器
const int FSR_OUTER = A1; // 外侧传感器

void setup() {
  Serial.begin(9600);
}

void loop() {
  int innerVal = analogRead(FSR_INNER);
  int outerVal = analogRead(FSR_OUTER);
  if ((innerVal - outerVal) > 350) { // 触发足内翻压力阈值
    triggerAlert(); // 隐性振动反馈与数据同步
  }
  delay(50);
}
```

- **柔性软体验证**：引入 **ThermoFit 热成型智能超材料技术**，将柔性传感面料内嵌于织物中，通过硬体验证保证高频行走下的**电子连续性与抗疲劳寿命**。

## 质感交响：医用硅胶与热成型面料的触觉重塑

> 外刚内柔的双层异质拼接，用蓝粉渐变的视觉张力消解医疗设备的冰冷符号。

![蓝粉渐变卡扣细节工艺](https://portfolio-image-cdn.zongxz66.workers.dev/project-1/buckle-detail.jpg)

- **刚性矫正骨架**：外层采用高强度 **医用硬质硅胶外壳**，精准覆盖脚踝与足背的核心受力区域；提供强效**力学支撑**并维持行走姿态，强力扭转肌肉痉挛导致的形变。

- **感官贴合内衬**：内层铺设柔软织物内置 **ThermoFit 传感面料**；具备优秀的**热成型自适应能力**，紧密贴合足底轮廓的同时，确保穿着时**空气流通与极佳的缓震透气性**。

- **潮流视觉锚点**：配备 **3 条可调节浅灰色绑带**，搭配两组**蓝粉渐变微交互卡扣**；主色调采用低饱和度的**浅蓝灰色**，通过时尚运动风的线条彻底抹除"残疾暗示"。

## 迭代轨迹：从草图阵列到 AIGC 设计具象化

> 经历数十次手绘骨骼推演与前沿人工智能的碰撞，打磨出最优的流线型仿生形态。

- **多维视角草图**：从骨骼解剖学出发，手绘多组侧面、背面及局部卡扣的**草图演变轨迹**；严密推演绑带围度调整对固定效果的实际影响。

- **人工智能破局**：在草图雏形基础上，引入 **AIGC 进行高频视觉迭代**；探索多组不同的流线型运动线条与网面拼接方案，快速逼近最佳的形态解。

- **工程量产对齐**：最终方案通过精确的**分件设计与模块化组装规划**，将硬质硅胶、传感织物与绑带卡扣拆解为可单独注塑与缝制的独立单元，完美兼顾**临床定制的高效性与量产可行性**。
