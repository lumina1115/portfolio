## 一、设计调研 —— 为什么做？

> 全球披萨炉市场 2025 年约 20–26 亿美元，亚太 CAGR >11%。中国披萨门店突破 6 万家，四年增长 60%。但现有产品无一触及"温润家居"象限。

**双场景痛点**
- **商用**：60%+ 门店投诉源于烤箱不稳定；出品高度依赖师傅经验，连锁扩张难复制
- **家用**：约 45% 用户操作复杂、火候难掌控；传统设备缺乏自适应调节与远程监控

**竞品空白扫描**

| 品牌 | 价格 | 核心短板 |
|------|------|----------|
| Ooni | ¥4000–6000 | 工业感强、户外属性显著 |
| Ninja | ¥2900 | 多功能但形态粗犷 |
| THOMSON | ¥1550 | 功能基础、无智能扩展 |

→ 共同盲点：外观硬朗、强调户外感，**"家用 × 温润自然"象限完全空白**。

## 二、设计定位 —— 做成什么样？

> "意承古窑，温润入家"——粗犷工业设备 → 具有"温润感"的家庭陪伴者。

**三大设计支柱**
- **有机美学**：借鉴传统窑炉浑圆形态，去机械线条堆砌
- **智能交互**：技术"低干预"，复杂功能隐于无形
- **温润自然**：砂砾触感 × 原木把手，材质即叙事

**目标用户 · Iris 画像**：28 岁 / 都市白领 / 上海独居 / 小红书重度用户。核心诉求：一键操作 · 快速烘烤 · 小巧便携 · 一机多用。

**产品定义**：不止 450°C 性能机器 → 家居空间中有"呼吸感"的艺术品。为追求仪式感的中产家庭，打造"慢生活"社交中心。

## 三、设计展开 —— 怎么做出来的？

> 将顶部"烟囱"从纯粹排热结构，重新定义为情感状态界面——用户通过烟囱亮度直觉感知炉温，无需盯屏。

**形态推导路径**

草图 → 穹顶 + 视窗 + 控制板分区确立 → 局部迭代深化

**关键节点决策记录**

| 节点 | 方案 | 决策依据 |
|------|------|----------|
| 烟囱 | 有机气泡造型 | 保留窑炉语意符号 |
| 把手 | 原木棒型 | 强化温润触感与抓握仪式感 |
| 翻盖 | 下翻式 ✓ | 铰链受力稳定，翻开后延伸形成操作台面 |
| 底座 | 四只圆柱形支脚 | 契合有机形态语言，兼顾隔热防烫 |

**翻盖三方案对比评估**
- 方案 A · 下翻式 ✓（铰链受力稳定，翻开后延伸形成操作台面）
- 方案 B · 上翻式（翻盖高度占位大，干扰上方空间）
- 方案 C · 左右对开（结构复杂，密封性弱）

**结构技术落地**
- 多区智能温控 + 上下独立环形加热管 + 温度传感器
- 五层分解：穹顶外壳 → 隔热内胆 → 智能主机层 → 加热控温层 → 陶瓷釉石板
- 扁形腔体：缩短热辐射路径，聚温效率提升
- 陶瓷釉石板：加厚设计，模拟传统窑烤蓄热质感

**★ 情感交互：温度指示灯渐变系统**

将顶部"烟囱"从纯粹排热结构，重新定义为**情感状态界面**：
- 炉温低 → 暗哑琥珀光
- 炉温中 → 稳定暖橙光
- 炉温高 → 明亮火焰光

功能逻辑 → 窑炉升温仪式感的转化：用户通过"烟囱亮度"直觉感知炉温，无需盯屏，将智能传感数据翻译为一种**可感知的身体经验**。

## 四、设计呈现 —— 最终结果如何？

**产品规格**：整机 420 × 380 × 360mm（紧凑台面级）；最高 450°C；5 大烹饪模式：烘焙 / 牛排 / 空炸 / 烧烤 / 上管烤。

**交互 UI · 三态流程**

```
预热中（实时温度 → 目标温度进度条）
    ↓
烹饪中（450°C 恒温 + 剩余时间倒计）
    ↓
完成（烘烤完成提示）
```

界面语言：黑底 + 琥珀橙字 → 与温度指示灯情感逻辑形成**视觉系统闭环**。

**★ 系统化 CMF 配色策略**

不止"选三款颜色"，而是以材质 + 色调构建**完整情感光谱**，每款配色对应差异化消费人群：

| 色系 | 材质语言 | 情感定位 | 目标人群 |
|------|----------|----------|----------|
| **玄武岩** | 哑光黑 + 深棕原木 | 专业蓄热感 · 性能人设 | 进阶烹饪爱好者 |
| **云砂白** | 米白砂质 + 浅色木 | 北欧温馨感 · 家用主力 | 核心目标用户 |
| **星尘砾** | 水磨石纹理 + 白桦木脚 | 艺术个性 · 高溢价定位 | 注重格调的品位用户 |

三款配色外壳统一保留"砂砾触感"，强化手工窑炉的**跨感官叙事**一致性。

**配件体系**：陶瓷披萨石（320×320mm）· 不锈钢烤网 · 铁镀铭篮 · 披萨铲 · 说明书
